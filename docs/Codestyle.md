# Code Style

## Prettier

- Runs formatter on save

## NativeWind class sorting

- Use `prettier-plugin-tailwindcss` to keep class name orders consistent
- It will work with NativeWind, it just sorts any tailwind like classes

## Imports

- Sort imports with `@trivago/prettier-plugin-sort-imports`

## File Names

- React components: `PascalCase.tsx`
- Hooks + Utilities: `camelCase.ts`
- Folders: `example`

## Commit Messages

- Use meanigful commit messages prefixed like `feat:`, `bug-fix:`, `test:`, ...
  - ex: `feat: add event list screen`, `fix: corrected /health response format`
