# Team Workflow

## Trello Board

#### Lists include:

- Backlog
- Sprint To Do
- In Progress
- In Review
- Done

#### Labels:

- Frontend
- Backend
- DevOps/Infrastructure
- QA/Test
- PM

#### Usage

- When picking up tickets, move it to in progress so team knows its currently being worked on.
- Move to In Review when PR has been made and link it in the comments.
- Once code compiles + tests have passed + and PR was reviewed and approved, move card to Done.
